Chapter 28

Reinforcement Learning

Phase 4: The model generates multiple answers, gets judged, and learns from the results — trial and error at scale.

SFT showed the model examples of good responses. But there are infinitely many possible responses to any prompt. Reinforcement Learning (RL) lets the model explore, generate its own responses, get feedback on quality, and improve.

How RL Training Works

① Give model a prompt (e.g., a math problem)
↓
② Model generates multiple responses (e.g., 8 different attempts)
↓
③ Judge each response: correct ✅ or wrong ❌
↓
④ Increase probability of correct responses, decrease wrong ones
↓ repeat thousands of times

Using medium temperature (Chapter 9) during generation ensures variety — some attempts will be correct, others wrong. The model learns from the contrast between them.

Tamil Math Example

🧮
RL in Action

Prompt: "ஒரு கடையில் 5 ஆப்பிள்கள் ₹120 என்றால், 8 ஆப்பிள்கள் எவ்வளவு?"
(If 5 apples cost ₹120 at a shop, how much do 8 apples cost?)

Response A ✅ Correct

Step 1: Price per apple = ₹120 ÷ 5 = ₹24
Step 2: Cost of 8 apples = ₹24 × 8 = ₹192

→ Reward signal: increase probability of this reasoning pattern

Response B ❌ Wrong

The cost would be ₹120 + 3 more apples = ₹120 + ₹72 = ₹192

Wait — this one also got ₹192 but with wrong reasoning (got lucky). A better judge would check the reasoning, not just the answer.

Response C ❌ Wrong

8 apples cost ₹120 × 2 = ₹240

→ Penalty signal: decrease probability of this reasoning pattern

GRPO: Group Relative Policy Optimization

The specific RL algorithm used is called GRPO. Instead of judging responses against an absolute standard, it compares responses within a group: correct ones get positive reward, wrong ones get negative reward, relative to the group average. This is more stable than absolute rewards and works well for math and code where correctness can be automatically verified.

For subjective tasks (creative writing, helpfulness), a reward model — another AI trained on human preferences — judges the quality of responses instead of checking for objective correctness.

Key Insight

Reinforcement learning lets the model learn from its own attempts. Generate multiple responses, judge them, reinforce the good ones. This is especially powerful for math and code where correctness is verifiable. Combined with SFT, RL is what makes modern AI assistants genuinely skilled at reasoning, problem-solving, and following complex instructions.
