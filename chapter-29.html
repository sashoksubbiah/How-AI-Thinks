You: Write a Python function to check if a number is prime.
AI:
def is_prime(n): if n < 2: return False for i in range(2, int(n**0.5) + 1): if n % i == 0: return False return True

The model isn't perfect — it's small (1.5 billion parameters) compared to production models (hundreds of billions). It will make mistakes, especially on complex reasoning or obscure knowledge. But the architecture, the training procedure, and the fundamental mechanisms are exactly the same as what powers ChatGPT, Claude, Gemini, and every other modern AI assistant. They're just bigger.
